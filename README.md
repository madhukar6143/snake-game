# 🐍 Snake Game

Built with Claude Code in a single session.

## Play
Open `snake.html` in any browser — no setup needed.

## Controls
- **Arrow Keys** — move snake
- **Space** — pause/resume
- **Enter** — restart after game over

## High Scores
Global high scores stored via JSONBin.io.

To set up your own JSONBin:
1. Sign up at jsonbin.io
2. Create a new bin with `{"highScore": 0}`
3. Replace `JSONBIN_BIN_ID` and `JSONBIN_API_KEY` in snake.html

## Built With
- Claude Code (single prompt)
- Vanilla HTML/CSS/JS
- JSONBin.io for persistent high scores
